interpshapefile

Interpolates values at points based on a shapefile data structure

Syntax

value = interpshapefile(ShapeStruct, lat, lon, attribute)

Description

value = interpshapefile(ShapeStruct, lat, lon, attribute) determines the attribute value at the given points defined by the matrices lat and lon based on the data in the geographic data structure ShapeStruct (created using the shaperead function).

If the specified attribute holds a numeric scalar, value will be a numeric array; points located outside the polygons will receive a NaN. Otherwise, value will be a cell array; points located outside the polygons will hold an empty array.

Remarks

This function assumes that the geographic region covered by the polygons is small enough that the lines connecting the points can be approximated using straight lines, and that the region does not cross the poles.

Also, this function was written to deal with shapefiles that hold only polygons that fit together to form an entire coverage area. It should work with shapefiles that also contain non-polygon objects, but has not been tested.
