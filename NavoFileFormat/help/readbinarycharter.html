readbinarycharter

Reads data from a binary charter file

Syntax

Data = readbinarycharter(filename)

Description

Data = readbinarycharter(filename) reads the data in filename into a the structure Data. A binary charter file is used to store gridded datasets at the Naval Oceanographic Office.

Output Structure

Field NameField Contents
westLonWestern boundary of data grid, in decimal degrees
eastLonEastern boundary of data grid, in decimal degrees
southLatSouthern boundary of data grid, in decimal degrees
northLatNorthern boundary of data grid, in decimal degrees
gridIntervalGrid interval, in minutes
widthWidth of grid (# cells)
heightHeight of grid (# cells)
endianValue66051
minimumValueMinimum value in the data grid
maximumValueMaximum value in the data grid
dataGridGrid of values, with the southernmost data in row 1 (top) and the westernmost data in column 1 (left)
