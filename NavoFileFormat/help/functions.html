Navo File Format Toolbox Functions

HFEVA Sediment Shapefiles

Most of NAVO's sediment data is stored in ESRI shapefiles and use the HFEVA sediment classification system.

  • loadhfevashape
  • plothfevashape
  • interpshapefile
  • hfevaTable
  • hfevaconvert
  • hfevaColorbar
  • hfevashapearea
  • plotshapefile
  • shapefilearea

Charter files

Charter files are often used at NAVO to store gridded datasets, e.g. bathymetry (*.fin files) and sediment (OAML database).
  • readbinarycharter
  • writebinarycharter
  • readcharterheader
  • loadoamlsed
  • plotcharter

MOODS database files

The MOODS database is part of the NAVO Data Warehouse; it provides sound speed profiles.
  • loadmoodssvp

Unisips Files

Unisips files store sidescan sonar data.
  • readunisips
  • plotunisips
  • readunisipsheader

ISS-60 survey files

ISS-60 stores survey lines and waypoints in .srv files.
  • readsrv
