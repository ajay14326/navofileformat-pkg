hfevaColorbar

Add a HFEVA colorbar to the current figure

Syntax

hfevaColorbar(positionVector, fontSize)
hfevaColorbar(positionVector, fontSize, indices)
h = hfevaColorbar(...)

Description

hfevaColorbar(positionVector, fontSize) adds a vertical colorbar showing all the HFEVA sediment types, where positionVector is a vector indicating [left bottom width height] of the colorbar with respect to the current figure. Each color is labeled by HFEVA standard sediment type, using the font size indicated by fontSize.

hfevaColorbar(positionVector, fontSize, indices) adds a colorbar with only the sediments indicated by the vector indices. See hfevaTable for all available sediment types and their indices.

h = hfevaColorbar(...) returns the handle of the colorbar.

Remarks

This colorbar is for labeling purposes only; it has no effect on the colormap of any axes that may be present in the figure.

Example

figure('Position', [50 50 200 500], 'color', 'w');
hfevaColorbar([.1 .1 .8 .8], 8)
