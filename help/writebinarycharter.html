writebinarycharter

Creates a binary charter file from a gridded dataset

Syntax

writebinarycharter(outputFile, westLon, eastLon, southLat, northLat, gridInterval, dataGrid)

Description

writebinarycharter(outputFile, westLon, eastLon, southLat, northLat, gridInterval, dataGrid) creates a binary charter file outputFile from the gridded dataset dataGrid. The variables westLon, eastLon, southLat, northLat, and gridInterval are used to fill in header information.
