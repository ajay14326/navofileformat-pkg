plothfevashape

Plots the polygon in a HFEVA shapefile

Syntax

plothfevashape(filename)
plothfevashape(filename, north, south, east, west)
handles = plothfevashape(...)

Description

plothfevashape(filename) plots the polygons in a HFEVA sediment shapefile to a mercator projection, where filename is either the full name or base name of the shapefile components.

plothfevashape(filename, north, south, east, west) sets the map boundaries to the indicated latitudes and longitudes (if not included, the bounding box of the polygons is used).

handles = plothfevashape(...) returns a structure holding the handles of the figure, map plot, map axis, and colorbar.
