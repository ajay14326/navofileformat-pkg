loadhfevashape

Loads a shapefile containing HFEVA sediment polygons

Syntax

HfevaStruct = loadhfevashape(filename)

Description

HfevaStruct = loadhfevashape(filename) loads a N x 1 geographic data structure from a HFEVA sediment shapefile. See hfevaTable for information about the HFEVA sediment classification system.

Output Structure

Field NameField Contents
Geometry'Polygon'
Lonvector a polygon vertex longitudes, NaN-delimited for multipart polyons
Latvector a polygon vertex latitudes, NaN-delimited for multipart polyons
CATHFEVA sediment category
typeIdHFEVA type ID
rmzRMZ (mean grain diameter)
color[red green blue] 0-255
patchFace, patchVerticesFace-vertex matrices for the polygon (for plotting purposes)
otherAny other fields included in the original shapefile
