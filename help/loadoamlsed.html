loadoamlsed

Loads an OAML sediment charter file

Syntax

Data = loadoamlsed(filename)

Description

Data = loadoamlsed(filename) loads the OAML charter file filename into the m x 1 structure Data. The output structure is of the same format of that returned by readbinarycharter. However, the grid is broken into several smaller 200 x 200 element grids (unless it is smaller than this to begin with) and the data values are converted to RMZ values.
