hfevaconvert

Convert between HFEVA values based on a reference table

Syntax

newvalue = hfevaconvert(value, convertfrom, convertto)

Description

Converts the matrix value between values available in the HFEVA sediment classification system. The following conversions are possible:

convertfrom'typeid', 'label', 'cat', or 'rmz'
convertto'typeid', 'label', 'cat', 'rmz', or 'color'

If multiple conversions are possible (e.g. going from rmz to cat), only the last value in the table is returned. If convertto is either 'label' or 'color', newvalue will be a cell array with the same dimensions as value; otherwise, newvalue will be a numeric array.

Examples

Find the RMZ values associated with three sediment types.

x1 = hfevaconvert({'Rock'; 'Gravel'; 'Sand'}, 'label', 'rmz')

x1 =
    -7.0000
    -3.0000
    1.5000

Note that conversions are not always reversible when converting between properties that do not have a one-to-one ratio:

x2 = hfevaconvert(x1, 'rmz', 'label')

x2 =
    'Rock'
    'Pebble'
    'Medium Sand'
